# MLX Projects for Mac

This repository is a curated cookbook of notable MLX projects that run on Apple Silicon Macs.
It helps people quickly discover high-signal MLX tools, apps, and libraries by keeping the list organized and quality-filtered.

Platform note: MLX targets Apple silicon, so these are Mac-focused projects.

If you want to add your project, open a PR. Projects with some traction will be added.
